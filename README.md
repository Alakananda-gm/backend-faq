# FAQ Backend

## Installation
1. Clone the repository.
2. Run `npm install`.
3. Start MongoDB and Redis.
4. Run `npm start`.

## API Usage
- Fetch FAQs in English: `GET /api/faqs`
- Fetch FAQs in Hindi: `GET /api/faqs?lang=hi`
- Fetch FAQs in Bengali: `GET /api/faqs?lang=bn`

## Testing
Run `npm test` to execute unit tests.
